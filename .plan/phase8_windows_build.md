# Phase 8: Windows 构建

## 目标
生成可分发的 Windows 安装包。

## 验证清单

| 测试项 | 验证方法 | 预期结果 |
|--------|----------|----------|
| 构建成功 | `npm run build -- --win` | 生成 `.exe` 文件 |
| 安装 | 运行安装包 | 程序正确安装 |
| 启动 | 点击图标 | 应用正常启动 |
| 快捷键 | 按住右 Alt | 托盘图标变化 |
| 录音 | 长按录音 | 生成 WAV 文件 |
| 转写 | 说话后松开 | 输出识别文字 |
| 文字输入 | 识别完成后 | 文字输入到焦点应用 |

## 构建步骤

### 1. 执行构建

```bash
# 在 macOS/Linux 上构建 Windows 版本
npm run build -- --win
```

### 2. 检查输出

```bash
ls -la dist/
```

**预期输出**:
```
Kory Whisper Setup.exe
Kory Whisper-0.1.0.exe
```

### 3. 安装测试

1. 复制 `Kory Whisper Setup.exe` 到 Windows 虚拟机/电脑
2. 运行安装程序
3. 按照安装向导完成安装

### 4. 功能测试

#### 4.1 应用启动
- 点击桌面快捷方式或开始菜单
- 观察系统托盘区域是否出现图标
- 检查日志文件

#### 4.2 快捷键测试
1. 打开任意文本输入框
2. 按住 **右 Alt** 键

**预期**:
- 托盘图标变化（显示录音中状态）
- 日志显示 `[Main] Long press started - recording...`

#### 4.3 完整流程测试
1. 保持按住右 Alt
2. 对着麦克风说话
3. 松开右 Alt
4. 观察文字是否输入到焦点应用

## 验证标准

- [ ] `dist/Kory Whisper Setup.exe` 存在
- [ ] 安装过程无报错
- [ ] 应用启动正常
- [ ] 快捷键监听正常
- [ ] 录音功能正常
- [ ] 转写功能正常
- [ ] 文字输入正常

## 常见问题排查

### 问题 1: ffmpeg 找不到

**症状**: 录音启动失败

**排查**:
```bash
# 检查 extraResources 中是否包含 ffmpeg
ls -la resources/app/ffmpeg/
```

### 问题 2: Whisper 找不到

**症状**: 转写失败

**排查**:
```bash
# 检查 bin 目录
ls -la resources/app/bin/
```

### 问题 3: 权限问题

**症状**: 输入模拟失效

**解决**:
- 以管理员身份运行应用
- 在 Windows 设置中启用"桌面自动化"
