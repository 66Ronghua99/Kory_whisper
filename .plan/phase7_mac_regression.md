# Phase 7: 回归测试 (Mac)

## 目标
确保 Mac 功能不受影响。

## 验证清单

| 测试项 | 验证方法 | 预期结果 |
|--------|----------|----------|
| 应用启动 | `npm run dev` | 菜单栏图标出现，无报错 |
| 快捷键监听 | 按住右 ⌘ | 状态栏显示录音中 |
| 录音功能 | 长按右 ⌘ 录音 | 生成 WAV 文件 |
| 转写功能 | 说话后松开 | 输出识别文字 |
| 文字输入 | 识别完成后 | 文字输入到焦点应用 |

## 验证步骤

### 1. 应用启动

```bash
npm run dev
```

**预期**:
- 无报错信息
- 菜单栏出现 Kory Whisper 图标
- 日志显示 `[Main] ==================== Kory Whisper Ready ====================`

### 2. 快捷键监听

1. 打开任意文本输入框（Notepad、TextEdit 等）
2. 按住 **右 ⌘** 键

**预期**:
- 菜单栏图标变化（显示录音中状态 ●）
- 日志显示 `[Main] Long press started - recording...`

### 3. 录音功能

1. 保持按住右 ⌘
2. 对着麦克风说话
3. 松开右 ⌘

**预期**:
- 菜单栏图标变化（显示处理中 …）
- 日志显示 `[Audio] Recording saved: /tmp/kory-whisper-*.wav`
- 临时目录有 WAV 文件生成

### 4. 转写功能

**预期**:
- 日志显示 `[Whisper] Starting transcription...`
- 日志显示 `[Whisper] Result: <识别文字>`
- 菜单栏显示成功状态 ✓

### 5. 文字输入

**预期**:
- 识别的文字自动输入到焦点应用
- 菜单栏状态恢复

## 验证标准

- [ ] 应用启动正常
- [ ] 快捷键监听正常
- [ ] 录音功能正常
- [ ] 转写功能正常
- [ ] 文字输入正常
